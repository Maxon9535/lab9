Програмне забезпечення(IDE):

PhpStorm

Розумне середовище від відомої компанії JetBrains призначене для розробки на PHP, JavaScript, HTML та CSS і ідеально підходить для роботи з різними CMS: Drupal, Wordpress, Symfony, Joomla та багатьма іншими. Середовище розробки глибоко аналізує структуру коду, допомагаючи уникати помилок, а також підтримує бази даних та SQL. https://www.jetbrains.com/ru-ru/phpstorm/

CLion

І знову продукт JetBrains. CLion – ідеальне кросплатформне рішення для тих, хто працює на C та C++ (і не тільки). Розумний редактор, зручний генератор коду, статичний та динамічний аналіз, безпечний рефакторинг… Особливості даного середовища розробки можна перераховувати нескінченно.
